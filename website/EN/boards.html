Freedombone

Supported ARM boards

The following ARM boards are supported by the build system. If your board isn't listed here then you may still be able to install Freedombone using Armbian.

  • beagleboneblack
  • beaglebonegreen
  • beaglex15
  • cubieboard2
  • cubietruck
  • pcduino3
  • banana-pro
  • a20-olinuxino-lime
  • a20-olinuxino-lime2
  • a20-olinuxino-micro

The latest image builds are obtainable with dat and can be found here.
